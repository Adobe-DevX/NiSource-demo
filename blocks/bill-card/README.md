# Bill Card

Promotional account-billing block for the landing page concept.

## Authoring fields

- `eyebrow`
- `bill-label`
- `amount-due`
- `bill-cta-label`
- `bill-cta-link`

## Notes

- The block follows global design tokens from `styles/styles.css`.
- If either CTA link is omitted, the related button renders in a disabled state.
