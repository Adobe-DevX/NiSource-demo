# Billing Payment

Billing and payment action block based on the NiSource account dashboard design.

## Authoring fields

- `heading`
- `left-top-label`
- `left-top-status`
- `left-top-link`
- `left-top-help-link`
- `left-bottom-label`
- `left-bottom-status`
- `left-bottom-link`
- `left-bottom-help-link`
- `right-top-label`
- `right-top-status`
- `right-top-link`
- `right-top-help-link`
- `right-bottom-label`
- `right-bottom-status`
- `right-bottom-link`
- `right-bottom-help-link`

## Notes

- Labels accept basic rich text to emphasize highlighted words (e.g., `AutoPay`).
- Status supports `currency`, `check`, or empty.
- Styles follow `design.md` via `--nisource-*` and shared shape tokens.
